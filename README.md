# Employee Management API

Sistema de gestão de RH com controle de funcionários, departamentos, salários e status de contrato. Ideal para empresas que precisam centralizar informações de colaboradores.

## Tecnologias
Java 17 · Spring Boot 3.2 · Spring Data JPA · MySQL · Maven · Swagger/OpenAPI

## Funcionalidades
- Cadastro completo de funcionários com validação de CPF e e-mail duplicado
- Filtros por departamento, status e nome
- Atualização de status (ACTIVE, ON_LEAVE, TERMINATED)
- CRUD completo com validação de dados

## Como Executar
```bash
mvn spring-boot:run
# Acesse: http://localhost:8090/swagger-ui.html
```
